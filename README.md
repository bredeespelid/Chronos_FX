# Chronos_FX

This repository contains a single Jupyter notebook (`Chronos_Main.ipynb`) with the full set of **Chronos FX forecasting experiments** used in a master’s thesis. The notebook implements **monthly and quarterly walk-forward evaluation** for **EUR/NOK level forecasting**, including:
- price-only baselines,
- macro-financial variants (incl. exogenous extensions),
- robust benchmark comparisons (e.g., driftless random walk),
- regime-split performance tables,
- bootstrap robustness checks.
